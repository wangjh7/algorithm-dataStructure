一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为 “Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。

问总共有多少条不同的路径？

示例1
![示例1](https://assets.leetcode.com/uploads/2018/10/22/robot_maze.png)
- 输入：m = 3, n = 7
- 输出：28

示例2
- 输入：m = 3, n = 2
- 输出：3
- 解释：
  从左上角开始，总共有 3 条路径可以到达右下角。
  1. 向右 -> 向下 -> 向下
  2. 向下 -> 向下 -> 向右
  3. 向下 -> 向右 -> 向下
